# initialser
initials avatar

## Installation
    $ go get github.com/leonlau/initialser
    
``` go
import ("github.com/leonlau/initialser")
d:=initialser.NewDrawer(initialser.NewAvatar("joey"))
d.Draw() //image.Image

```
